## Deskripsi

Tugas 3 ini berisi implementasi konsep-konsep Linear Algebra menggunakan Python, NumPy, dan Matplotlib berdasarkan materi Practical Linear Algebra.

Setiap chapter disusun dalam bentuk Jupyter Notebook (.ipynb) yang memadukan:

- teori singkat
- implementasi Python
- visualisasi
- analisis hasil
- keterkaitan dengan Machine Learning dan Data Science

Tujuan utama tugas ini adalah memahami bagaimana konsep aljabar linear digunakan dalam komputasi numerik, machine learning, data science, computer vision, dan artificial intelligence.

## Cara Menjalankan Notebook

1. Buka Google Colab atau Jupyter Notebook.
2. Upload file notebook yang ingin dijalankan.
3. Jalankan setiap cell secara berurutan dari atas ke bawah.
4. Pastikan library NumPy dan Matplotlib sudah tersedia.
5. Perhatikan output, visualisasi, dan analisis pada setiap section.
